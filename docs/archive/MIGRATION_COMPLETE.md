### 1. Semantic Organization
- Organized by **concept** and **layer**, not arbitrary folders
- Each document semantically linked to related concepts
- Knowledge graph enables intelligent navigation

```

architecture/    → HOW it works (implementation)
concepts/        → WHAT it means (understanding)
```

- **Users** start at guides/
- **Contributors** start at contributing/
