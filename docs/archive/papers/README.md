## ðŸŽ“ Writing Guidelines

### **Structure** (8 pages for NeurIPS/ICML)
1. **Abstract** (150-200 words)
   - Problem, approach, results, impact

2. **Introduction** (1 page)
   - Motivation
   - Gap in SOTA
   - Contributions (numbered list)

3. **Related Work** (1 page)
   - Prior work
   - Comparison to our approach
   - What's novel

4. **Method** (2-3 pages)
   - Architecture
   - Algorithms
   - Training procedures

5. **Experiments** (2-3 pages)
   - Datasets
   - Baselines
   - Results (tables/figures)
   - Ablation studies

6. **Discussion** (0.5 page)
   - Analysis
   - Limitations
   - Future work

7. **Conclusion** (0.25 page)
   - Summary
   - Broader impact

8. **References** (1-2 pages)

### **Style**
- Clear, concise prose
- Avoid jargon
- Define all terms
- Use active voice
- Present tense for contributions

### **Figures/Tables**
- High quality (vector graphics)
- Clear labels
- Referenced in text
- Standalone captions
